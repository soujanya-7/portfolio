Budget Tracking System

A Simple Solution to Track Personal Budgets and Expenses

Project Overview

The Budget Tracking System is a tool designed to help individuals manage their personal finances effectively. It allows users to track their income, expenses, and set budget limits, helping them to stay on top of their finances and save for future goals.

The system provides real-time tracking, graphical representations of expenses, and budget categories. It allows users to input their monthly income and categorize their expenses for easier tracking.

Key Features

  • Track Income and Expenses
  • Set Monthly Budgets
  • Generate Graphical Reports
  • View Expense Categories
  • Real-Time Budget Monitoring

Technologies Used

  • Frontend: HTML, CSS, JavaScript
  • Backend: Python (Flask)
  • Database: SQLite
  • Version Control: Git

This project was created as a way to help people better manage their finances in a simple and user-friendly way. The goal is to provide an easy-to-use application with valuable features to improve personal budgeting.
